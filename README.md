# Rolling Popcorn 🍿

Rolling Popcorn is a high-performance movie ticket booking system built to handle concurrent seat selection and real-time theater discovery. 


🚀 Key Features


Concurrency Control: Implements a distributed seat-locking mechanism using Redis to prevent race conditions during high-traffic booking windows. 

Dynamic Discovery: Uses Geolocation APIs for auto-location detection to suggest nearby cinemas with sub-150ms latency. 

Multi-Role Management: Robust dashboard for Managers to handle bulk show adding and for Customers to manage preferences. 

Responsive Design: Pixel-perfect UI designed in Figma and implemented using React.js. 

🛠 Tech Stack


Frontend: React.js, Tailwind CSS 

Backend: Django (Python), Node.js 

Database: MySQL (Relational schemas for complex table connections) 

Caching/Concurrency: Redis 

Design: Figma
